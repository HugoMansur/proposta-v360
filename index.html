Protestos em cartório

Consultando o CNPJ, a gente vê protestos contra a empresa (empresa como devedora). Retorna quem protestou, o valor e a data. Empresa de R$ 800M com protesto de R$ 1.500 de transportadora = falta de processo, não de dinheiro.

Dados detalhados (credor + valor) confirmados via API pra SP. Outros estados retornam existência, sem detalhe.

IMPACTO POTENCIAL

Estimativas conservadoras — o piloto mede o impacto real

Mais SQLs por BDR

Hoje a conversão conta → SQL é 25%. De SQL pra Discovery já é 75% — o time converte bem quando chega lá. O gargalo está no topo. Se com priorização e dossiê a gente subir conta → SQL pra 30% (+5pp conservador), cada BDR gera +3 SQLs/mês.

60 contas × 5pp = +3 SQLs por BDR. Com 6 rampados = +18 SQLs/mês. Como 75% viram Discovery, são ~14 Discoveries a mais pro time.

Mais tempo ligando, menos pesquisando

Se o tempo de pesquisa cai de 40 min pra 5 min por conta, o BDR recupera ~35 horas/mês. São horas que voltam pra ligação, follow-up e qualificação — o trabalho que realmente gera SQL.

Hoje o BDR faz ~15 ligações/dia. Com menos pesquisa, pode chegar a 25-30.

Esses números são conservadores. O piloto de 30 dias com 2 BDRs vai medir o impacto real antes de qualquer compromisso. Você faz a conta com os números de verdade.
